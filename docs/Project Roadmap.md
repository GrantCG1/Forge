# Project Vision

To design and build Forge: a personal health companion that educates, adapts and assists people in living healthier, more balanced lives while reducing unnecessary cognitive load while preserving personal autonomy.

Forge is built upon a Christian philosophy of wisdom, stewardship, encouragement and service. Scientific evidence informs what Forge recommends. Christian principles shape how Forge supports people.

# Guiding Principles

- Build philosophy before features.
- Simplicity before sophistication.
- Behaviour before implementation.
- Knowledge before decisions.
- Education before automation.
- Long-term quality over short-term speed.
